# Course Enrollment Webpage

A simple, responsive webpage for displaying courses, collecting user information, and highlighting program benefits. This project demonstrates the use of HTML5 semantic elements like `<section>`, `<article>`, `<aside>`, `<footer>`, and form inputs.

---

## Features

- **Courses Section**: Lists multiple courses using `<article>` elements.
- **Program Highlight**: Side content with `<aside>` for “Why Join This Program?”
- **User Form**: Collects user data with inputs including dropdowns, email, and file uploads.
- **Footer**: Includes contact email and copyright information.
- **Semantic HTML**: Uses modern HTML5 tags for better accessibility and structure.
- **Responsive Ready**: Structure is ready for styling with CSS for different screen sizes.
